# Methodology

Methodology specifies how SignalAF moves from submitted telemetry to public measurements and bounded interpretations.

## Knowledge pyramid

- **Observations:** windowed raw telemetry and integrity evidence.
- **Metrics:** deterministic formulas applied to those pillars.
- **Signals:** qualified interpretations of metric patterns.
- **Research:** versioned claims that preserve data and method provenance.

Read the formulas with their guards and limitations. Methodological transparency does not convert a metric into a universal measure of performance.
